Welcome to the Flask + Docker Learning Project

A hands-on project to learn how to containerize a Python web application

What You'll Learn

  • How to build a simple web app with Flask
  • How to write a Dockerfile for Python apps
  • How to build and run Docker images
  • How Flask routing and templates work

Project Structure

flask-kube-app/
├── app.py              # Flask application
├── Dockerfile          # Docker build instructions
├── requirements.txt    # Python dependencies
├── static/
│   └── style.css       # Stylesheet
└── templates/
    ├── base.html       # Base template
    ├── index.html      # Home page
    ├── docker.html     # About Docker
    ├── flask.html      # About Flask
    └── how_it_works.html

Quick Start

# Build the Docker image
docker build -t flask-kube-app .

# Run the container
docker run -p 5001:5001 flask-kube-app

# Visit http://localhost:5001
