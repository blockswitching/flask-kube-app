About Flask

A lightweight Python web framework

What is Flask?

Flask is a micro web framework for Python. It's called "micro" because it keeps the core simple but extensible. It doesn't include a database layer, form validation, or other components — you pick what you need.

Core Concepts

Routes

URLs mapped to Python functions. When a user visits a URL, Flask calls the matching function.

@app.route('/')
def home():
    return render_template('index.html')

Templates

HTML files with placeholders that Flask fills in using the Jinja2 templating engine.

{% extends "base.html" %}
{% block content %}
  <h1>Hello!</h1>
{% endblock %}

Static Files

CSS, JavaScript, and images served from the static/ folder.

url_for('static', filename='style.css')

How This App Works

from flask import Flask, render_template

# Create the Flask application instance
app = Flask(__name__)

# Define a route for the home page
@app.route('/')
def home():
    return render_template('index.html')

# Run the app (for local development)
if __name__ == '__main__':
    app.run(host='0.0.0.0', port=5001)
